# ChainForge Frontend

Next.js 14 web interface for ChainForge — the AI-powered blockchain project builder.

## Features

- **AI Project Builder** — Describe your blockchain project in plain English
- **Live Preview** — See generated contract code in real-time
- **One-Click Deploy** — Deploy to supported chains directly from the UI
- **Wallet Integration** — Connect with RainbowKit/Wagmi
- **Contract Explorer** — View deployed contracts, ABIs, and transaction history

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Web3:** RainbowKit + Wagmi + Viem
- **State:** TanStack Query
- **Language:** TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Environment Variables

See the root `.env.example` for required environment variables. The frontend uses:

- `NEXT_PUBLIC_WALLET_CONNECT_PROJECT_ID` — WalletConnect Cloud project ID
- `NEXT_PUBLIC_CHAIN_ID` — Default chain ID (11155111 for Sepolia)

## Structure

```
src/

├── hooks/         — Custom React hooks
```
