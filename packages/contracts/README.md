# ChainForge Contracts — Foundry project for smart contract templates

This package contains the Foundry project setup for compiling, testing, and deploying smart contracts generated by ChainForge.

## Setup

```bash
# Install Foundry (if not installed)
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Install dependencies
forge install OpenZeppelin/openzeppelin-contracts
forge install foundry-rs/forge-std
```

## Usage

```bash
# Build contracts
forge build

# Run tests
forge test

# Deploy to Sepolia
forge script script/Deploy.s.sol --rpc-url $SEPOLIA_RPC_URL --broadcast --verify
```
